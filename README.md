
# 🌥️ Wispy – Cloud ERP Landing Page

A high-performance, responsive landing page built with **Next.js** and **Tailwind CSS**, optimized with **lazy loading** for enhanced speed and user experience.

---

## 🚀 Features

- ⚡ **Next.js** – Fast, SEO-friendly React framework
- 🎨 **Tailwind CSS** – Utility-first CSS framework for modern UI
- 🖼️ **Lazy Loading** – Images and sections load on demand
- 📱 **Responsive Design** – Mobile-first layout and styling
- 🌐 **SEO Ready** – Meta tags and structure designed for visibility
- 🧠 **Component-Based** – Easy to maintain and extend

---

## 🛠️ Tech Stack

| Technology      | Description                                  |
|-----------------|----------------------------------------------|
| **Next.js**     | React-based framework with SSR & SSG support |
| **Tailwind CSS**| Utility-first CSS for fast UI development    |
| **React**       | Frontend library for building UI components  |
| **Lazy Loading**| Native + IntersectionObserver API for performance |

## 📦 Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/milankatira/wispy-erp-landing
cd wispy-erp-landing
````

### 2. Install dependencies

```bash
npm install
# or
yarn install
```

### 3. Run the development server

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

---

## 📌 Build for Production

```bash
npm run build
npm run start
```
